如何把阿里云域名绑定到 Vercel 服务器

发表于2024-08-02|更新于2025-09-04|运维部署
|总字数:849|阅读时长:2分钟|浏览量:

本文介绍如何把阿里云域名绑定到 Vercel 服务器,让您的网站通过 Vercel 服务器托管。我们以阿里云购买的域名为例子（其他平台类似），下面是详细的步骤来完成这一配置。

修改阿里云域名的DNS

访问 阿里云官网 并登录到您的账户。点击右侧的控制台，在控制台主页中，点击左侧的展开选项按钮，选择云解析 DNS ，进入域名管理页面。如图所示：

在域名列表中找到您要配置的域名，并点击该域名的“解析设置”按钮，进入解析设置页面，如图所示：

进入到详情页面，我们添加如下记录，如图所示

添加或修改 A 记录

  • 在 DNS 解析设置页面，点击“添加记录”。
  • 记录类型：选择“A”记录。
  • 主机记录：输入主机名，通常为 @（表示根域名）或指定子域名（如 www）。
  • 记录值：输入 Vercel 提供的 IP 地址（如果 Vercel 提供了 IP 地址）。
  • TTL：选择或保持默认值（通常是 600 秒或 3600 秒）。
  • 点击“确认”保存记录。

添加 CNAME 记录

  • 在 DNS 解析设置页面，点击“添加记录”。
  • 记录类型：选择“CNAME”记录。
  • 主机记录：输入子域名（如 www）。
  • 记录值：输入 Vercel 提供的 CNAME 地址（例如 cname.vercel-dns.com）。
  • TTL：选择或保持默认值。
  • 点击“确认”保存记录。

检查 DNS 记录

  • 确保您添加或修改的记录正确无误，并且已保存。

如何在 Vercel 配置域名

  • 登录到您的 Vercel 控制台。
  • 进入您要配置的项目。
  • 在项目的设置中，找到“Domains”部分，点击“Add Domain”。
  • 输入您的域名，点击“Add”。
  • Vercel 会提供验证步骤，您可能需要在阿里云的 DNS 设置中添加相应的 TXT 记录以验证域名所有权。

解析

  • CNAME 记录 (cname.vercel-dns.com) 是指向 Vercel 的 DNS 记录，通常用于将自定义域名指向 Vercel 的服务器。

  • A 记录 (76.76.21.21) 是 Vercel 使用的公共 IP 地址，用于将流量路由到 Vercel 托管的网站。

如下图所示：

验证 DNS 记录

  • 完成所有记录添加后，等待 DNS 记录的传播，通常需要几分钟到48小时不等。
  • 返回到 Vercel 的域名设置页面，检查域名状态是否为“已验证”或“已连接”。

注意事项

  • DNS 记录传播时间：DNS 记录的更改可能需要时间才能生效，通常从几分钟到48小时不等。您可以使用在线 DNS 工具（如 whatsmydns.net）检查 DNS 记录的传播情况。
  • TTL 设置：如果您之前的 TTL 值设置得较高，DNS 记录的更改可能会更慢。可以适当调整 TTL 值，但一般来说，默认值也可以。
  • 域名设置在 Vercel 上：确保您在 Vercel 的域名设置中正确添加了域名，并且完成了验证步骤（如添加 TXT 记录进行域名验证）。
  • 域名解析的 NS 记录：如果您的域名的 NS（Name Server）记录指向了阿里云的 DNS 服务器，那么您在阿里云 DNS 控制台中配置的记录将会生效。确保您的域名 NS 记录设置正确（通常在域名注册商处设置）。
