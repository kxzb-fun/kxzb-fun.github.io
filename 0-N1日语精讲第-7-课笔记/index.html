日语精讲精练第7课重点总结

一、学习目标

  1. 掌握家庭情况的日语介绍方式，区分日语称谓的“内”“外”之别。
  2. 初步理解格助词「が」的使用方法。
  3. 能在句尾恰当使用语气助词「ね」和「よ」。

二、核心词汇

本课核心词汇分为会话、语法解释、习题三类，重点掌握N5/N4级基础词汇，以及家庭称谓、数量表达相关词汇，以下为高频重点：

词汇 假名 词性 含义 等级
昨日 きのう 名 昨天 N5
兄弟 きょうだい 名 兄弟；兄弟姐妹 N4
兄/弟 あに/おとうと 名 哥哥/弟弟 N5
写真 しゃしん 名 照片 N5
家族 かぞく 名 家属，家族 N5
両親 りょうしん 名 双亲 N5
いくつ いくつ 名 几岁，几个 N5
かわいい かわいい 形Ⅰ 可爱的 N5
妻/夫 つま/おっと 名 妻子/丈夫 N4
息子/娘 むすこ/むすめ 名 儿子/女儿 N4

三、核心语法

1. 名词句(敬体)的过去时

为名词谓语句的时态拓展，口语中「では」可替换为「じゃ」，活用规则如下：

时态 肯定形式 否定形式
现在/将来 N です。 N ではありません。
过去 N でした。 N ではありませんでした。
例句：弟はまだ小学生でした。/我弟弟那时还是小学生。

2. 动词句(敬体)的时态

以「ます」结尾构成敬体，「ます」为现在/将来肯定形式，活用规则以「来ます」为例：

时态 肯定形式 否定形式
现在/将来 V ます。 V ません。
过去 V ました。 V ませんでした。
例句：昨日、ジョージさんのお姉さんが来ました。/昨天乔治的姐姐来了。

3. 疑问词が「提示主语」

格助词「が」接名词后提示主语，疑问词作主语时，必须用「が」提示，回答也需对应使用「が」，不可用「は」。
例句：A:研究室に誰がいますか。/研究室里有谁？B:マイケルさんと先生がいます。/有迈克尔和老师。

4. 日语中的连体词

「この/その/あの/どの」为连体词，不能独立使用，必须后接名词（体言），属于「こそあど」系指示词汇。

  • 正确：これは私のパソコンです。/这是我的电脑。；このパソコンは私のです。/这台电脑是我的。
  • 错误：このは私のパソコンです。；これパソコンは私のです。

5. だけ「限定」

助词「だけ」接名词/数量词后表示限定，译为“只、仅仅”。
例句：朝ご飯はりんごだけでした。/早饭只吃了苹果。

6. N1 の N2「同位语」

「の」在此处表示前后名词所指相同，N1多为表关系/性质的名词，N2多为专有名词（如人名），区别于表所属/属性的「の」。
例句：弟のハチです。/这是我弟弟小八。

7. 句子よ。「主张、提示」

语气助词「よ」用于句尾，表说话人的主张、断定，语气较强，口语中多为升调，用于提示对方注意信息。
例句：10年前の写真ですよ。/是10年前的照片哦。

四、表达扩展

1. 日语家族称谓：内外有别

日语亲属称谓需严格区分谦称、家庭内部称呼、敬称，谦称用于向外部介绍自己亲属，敬称用于称呼对方亲属，核心称谓对照如下：

亲属 谦称（自谦，对外说自己亲属） 家庭内部称呼 敬称（尊敬，称对方亲属）
爸爸 父（ちち） お父さん（おとうさん） お父さん（おとうさん）
妈妈 母（はは） お母さん（おかあさん） お母さん（おかあさん）
哥哥 兄（あに） お兄さん（おにいさん） お兄さん（おにいさん）
弟弟 弟（おとうと） - 弟さん（おとうとさん）
妻子 妻（つま）/家内（かない） - 奥さん（おくさん）
丈夫 夫（おっと）/主人（しゅじん） - ご主人（ごしゅじん）
爷爷 祖父（そふ） おじいさん おじいさん
奶奶 祖母（そぼ） おばあさん おばあさん

补充：对方的家人用「ご家族」表示，为汉语词前加接头词「ご」表敬称，类似用法还有「ご意見」「ご住所」。

2. 家庭人数的表达

（1）询问/描述句式

  • 句式1：(～さんは)何人家族ですか？/～さんのご家族は何人ですか？（你家有几口人？）
  • 句式2：私の家族は～人です。/(私は)～人家族です。（我家有～口人。）

（2）「人」的特殊读音

数量词「人」的读音为不规则变化，重点记忆：
1人（ひとり）、2人（ふたり）、3人（さんにん）、4人（よにん）、5人（ごにん）、疑问词「何人（ななにん）」。

3. 日语年龄的表达

（1）询问句式

敬体询问：おいくつですか？（您几岁？）；「いくつ」为和语词，前加「お」表敬意。

（2）年龄的构成与读音

由「数字+歳（さい）」构成，重点特殊读音：
1歳（いっさい）、8歳（はっさい）、10歳（じっさい）、20歳（はたち，特殊表达，无需加「さい」）。

4. 常用口语表达

  1. いえ：表否定，比「いいえ」更口语化。
  2. そして：连接句子/单词，表递进、累加，译为“而且、还、接着”。
  3. ～君（くん）：接在年纪比自己小的男性名字后，表亲密，比「～さん」随意；职场中男性上司也可用于称呼年轻女下属；可用于对宠物的称呼（如ハチ君）。
  4. かわいい：形容词，表“可爱的、漂亮的”，修饰人/事物。

五、重点句式（日常交际/家庭介绍）

  1. ～がいます。/有～（人）。例：兄と弟がいます。/我有一个哥哥和一个弟弟。
  2. ～は～にいます。/～在～（地方）。例：兄はアメリカにいます。/我哥哥在美国。
  3. これが～です。/这是～（人）。例：これが弟です。/这是我的弟弟。
  4. ～年前/昨日/去年，～でした/ました。/～前/昨天/去年，～（过去的状态/动作）。例：10年前の写真です。/这是十年前的照片。
