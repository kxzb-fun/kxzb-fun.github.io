梁爽老师日语精讲精练 第8课 笔记

学习目标

  1. 掌握点餐、购物场景的礼貌日语表达
  2. 熟练运用100以上数字及日语数量词

核心会话场景

快餐店点餐：询问美食广场位置→点芝士汉堡+热咖啡（指定S码）→结账付款→追加购买苹果派，涵盖点餐、选规格、付款、追加点餐全流程。

重点单词

场景核心词

日语 词性 释义 等级
店員(てんいん) 名 店员 N4
お客様(おきゃくさま) 名 顾客（敬称） N4
いらっしゃいませ - 欢迎光临 N3
会計(かいけい) 名/他Ⅲ 结账、付款 N2
お返し(おかえし) 名 找零 N1
ポイントカード 名 积分卡 -
アリペイ 专 支付宝 -
サイズ 名 尺码、杯型 N4

数量词/数字相关

日语 词性 释义 等级
一つ(ひとつ) 名 一个 N5
百(ひゃく) 名 百 N5
十(じゅう) 名 十 N5
円(えん) 名 日元 N5
枚(まい) 名 张、件（量词） N5
本(ほん) 名 根、支（量词） N5
台(だい) 名 台、辆（量词） N5

核心语法

1. N を ください

  • 用法：提示宾语，表“请给我……”，点餐/购物专用
  • 例句：チーズバーガーを一つください。（请给我一个芝士汉堡）
  • 拓展：表“还要……”用「も」替换「を」，如：ホットコーヒーもください。（还要热咖啡）

2. N を (数量词) ください

  • 用法：数量词放「ください」前，表“要……数量的……”
  • 例句：入場券を3枚ください。（请给我三张入场券）
  • 规则：日语数量词一般置于动词前

3. N で お願いします

  • 核心用法1：表方式/规格，点餐/购物/付款时用，替代「ください」更礼貌
    • 例句：Sサイズでお願いします。（要小杯的）；アリペイでお願いします。（用支付宝付款）
  • 核心用法2：表状态，前文已提及物品时，用此句式强调规格，更地道
  • 拓展：可用于其他场景，如：Eメールでお願いします。（请发邮件联系）

4. N で ございます

  • 用法：「です」的郑重语，服务行业/正式场合使用
  • 规则：主语一般为自己/己方的人/物，不用于对方
  • 例句：エスカレーターは左側でございます。（自动扶梯在左侧）

5. どうなさいますか

  • 用法：服务业专用，「する」的敬语形式，表“您要选哪种？”
  • 例句：ホットコーヒーのサイズはどうなさいますか。（咖啡您要什么杯型？）
  • 辨析：「〜はどうですか」用于询问对方看法，如：このコーヒーはどうですか。（这个咖啡怎么样？）

重点表达

1. 百以上数字读音规则

  • 百：300→さんびゃく、600→ろっぴゃく、800→はっぴゃく
  • 千：3000→さんぜん、8000→はっせん
  • 组合：110→ひゃくじゅう、1800→せんはっぴゃく

2. 常用数量词用法

量词 适用对象 特殊读音 疑问词
〜つ 十以内物品（个） 六→むっつ、八→やっつ いくつ
〜個 不限数量（个） - なんこ
〜枚 扁平物品（张、件） - なんまい
〜本 细长物品（根、支） 1→いっぽん、3→さんぼん なんぼん
〜台 机器/交通工具（台、辆） - なんだい

3. 购物/点餐/付款三步表达

  1. 陈述需求：Nをください / Nをお願いします（后者更礼貌）
  2. 补充规格/方式：Nでお願いします（选规格、包装、支付方式）
  3. 结账付款：お会計、お願いします；現金/カードでお願いします

4. 常用回应语辨析

表达 语气 适用场景
わかりました 普通礼貌 日常回应对方要求
かしこまりました 郑重礼貌 服务行业对顾客、下级对上级
はい 简洁 递东西时提醒对方，如递钱/名片
大丈夫です 委婉 拒绝对方提议，如：ポイントカードは大丈夫です。（不用积分卡了）
