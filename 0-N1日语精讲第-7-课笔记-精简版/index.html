日语学习笔记：第7课 - スマホに写真がありますよ

一、 学习目标

  1. 学会介绍家庭成员，掌握日语称谓中“内”与“外”的区别。
  2. 掌握时态变化：学习名词谓语句和动词谓语句的敬体过去时表达。
  3. 理解助词用法：重点学习助词「が」提示主语以及语气助词「ね」和「よ」的用法。

二、 重点语法详解

1. 敬体句的时态变化

日语中，名词句和动词句的过去时态有特定的变化规律。

句型类别 时态 肯定形式 否定形式
名词句 现在/将来 N です N ではありません
过去 N でした N ではありませんでした
动词句 现在/将来 V ます V ません
过去 V ました V ませんでした
  • 口语注意：「では」在口语中常替换为「じゃ」。
  • 例句：
    • 昨日は日本語の試験でした。（昨天有日语考试。）
    • ジョージさんのお姉さんが来ました。（乔治的姐姐来了。）

2. 疑问词 + が（提示主语）

  • 用法：格助词「が」用于提示主语。当疑问词（如：谁、哪个）做主语时，必须用「が」而不能用「は」。回答时也同样用「が」提示。
  • 例句：
    • A：研究室に谁がいますか。（研究室里有谁？）
    • B：マイケルさんがいます。（有迈克尔。）

3. 指示连体词：この、その、あの、どの

  • 特性：属于“连体词”，不能独立使用，后面必须紧接名词（体言）。
  • 对比：
    • 正确：このパソコンは私のです。（这台电脑是我的。）
    • 错误：これパソコンは……（「これ」是代词，不能直接接名词）

4. 助词「だけ」与「の」（同位语）

  • 「だけ」：接在名词或数量词后，表示限定（只、仅仅）。
    • 例句：アジア人は、私だけです。（亚洲人只有我一个。）
  • 「の」作为同位语：连接两个指代相同对象的名词，通常前项表示关系/性质，后项为具体名称。
    • 例句：弟のハチです。（这是我弟弟，小八 [小狗]。）

三、 表达扩展：家族称谓与人数

1. 称谓的“内外之别”

日语中介绍家人时，必须区分自谦（内）和尊称（外）。对别人介绍自己家人用谦称，称呼对方家人用敬称。

成员 对外介绍自己家人（谦称） 称呼对方家人/家内直呼（敬称）
爸爸 父（ちち） お父さん（おとうさん）
妈妈 母（はは） お母さん（おかあさん）
哥哥 兄（あに） お兄さん（おにいさん）
姐姐 姉（あね） お姉さん（おねえさん）
弟弟 弟（おとうと） 弟さん（おとうとさん）
妹妹 妹（いもうと） 妹さん（いもうとさん）

2. 人数与年龄的表达

  • 人数（-人）：注意 1人 (ひとり) 和 2人 (ふたり) 的特殊读音，4人读作 よにん。
  • 年龄（-歳）：
    • 询问：おいくつですか。（您几岁？）
    • 特殊读音：1岁 (いっさい)、8岁 (はっさい)、10岁 (じゅっさい)、20岁 (はたち)。

四、 语气助词「ね」与「よ」

  • 「ね」：用于句尾，表示向对方确认或征求赞同。
  • 「よ」：用于句尾，表示说话人的主张、断定或强调（通常读升调），用于提示对方不知道的信息。
    • 例句：10年前の写真ですよ。（是10年前的照片哦。）
