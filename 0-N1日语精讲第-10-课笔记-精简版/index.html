日语学习笔记：第10课 - 映画の約束

一、 学习目标

  1. 掌握形容词及名词谓语句的敬体过去时活用形式。
  2. 学习形容词的中顿形式，并能用多个形容词描述人或事。
  3. 学会发出邀请（一起做事）及如何礼貌地回应他人的邀约。

二、 重点语法详解

1. 形容词及名词谓语句的“过去时”

根据形容词分类，过去时的变化规律如下：

词性 时态 肯定形式 (敬体) 否定形式 (敬体)
Ⅰ类形容词 (例：高い) 过去时 ～かったです ～くありませんでした (或 ～くなかったです)
Ⅱ类形容词 (例：静か) 过去时 ～でした ～ではありませんでした (或 ～じゃなかったです)
名词 (例：学生) 过去时 ～でした ～ではありませんでした (或 ～じゃなかったです)
  • 注意：形容词「いい」的过去肯定形式为 「よかったです」。
  • 口语替换：在否定句中，「では」在口语中通常替换为**「じゃ」**。

2. 形容词的中顿（连用形）

当需要用两个或更多形容词描述同一事物（并列关系）时，前项需使用中顿形式。

  • Ⅰ类形容词：将词尾「い」变为 「くて」。
    • 例：主人公は背が高くて、魅力的です。（主人公又高又有魅力。）
  • Ⅱ类形容词：在词干后加 「で」。
    • 例：昨日の試験問題は複雑で、难しかったです。（昨天的考题又复杂又难。）
  • 要点：中顿形式只由前项词性决定，不受后项词性影响；时态由句尾谓语决定。

3. 程度副词：全然（ぜんぜん）

与「あまり」类似，用于加强否定语气，表示“完全（不）……”或“一点也（不）……”。

  • 例：全然难しくなかったですよ。（一点都不难哦。）

4. 助词用法

  • N まで：表示时间或地点的终点，意为“到……为止”。
  • N とか：用于列举例子，多用于口语。使用时常省略「は」「が」「を」等助词。

三、 时间与邀约表达

1. 时刻的表达

  • 几点：数字 + 時（じ）。注意：4点 (よじ)、7点 (しちじ)、9点 (くじ) 的读音。
  • 几分：数字 + 分（ふん/ぷん）。注意：10分 (じっぷん)、30分 (さんじっぷん) 等促音变化。
  • 半点：数字 + 時半（じはん）。

2. 提议与邀请

  • 提议方式 1：N はどうですか（……怎么样？），用于建议时间、内容或选项。
  • 提议方式 2：（いっしょに）N をしましょうか（[一起]……吧？），接在具有动作意义的名词后发出邀请。

3. 回应邀约

  • 赞成：いいですね（好啊）、そうしましょう（就这么办吧）、はい、大丈夫です（好的，没问题）。
  • 拒绝：通常伴随歉意，使用 すみません。ちょっと……（不好意思，有点[不方便]……），随后可提议下次：今度はぜひ一緒に（下次请务必一起）。

四、 常用口语及感叹词

  • よかったですね：表示为对方发生的好事感到高兴（太好了，真好呀）。
  • まあ、なんとか：表示“勉强还可以”、“总算……”（回答进展时使用）。
  • うん / ううん：分别是「はい」和「いいえ」的口语形式，用于亲密关系。
  • あら：多为女性使用，表示惊讶的感叹词（哎呀）。
