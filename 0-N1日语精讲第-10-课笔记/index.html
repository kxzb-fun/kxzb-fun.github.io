梁爽老师日语精讲精练 第10课 笔记

学习目标

  1. 掌握形容词谓语句敬体过去时的活用形式，能用形容词描述过去的人/事
  2. 学会日语邀约的表达及对邀约的赞成/拒绝回应

核心会话场景

日常邀约：聊昨日电影观感→询问考试情况→邀约下周一起看电影→商定时间（周四下午4点半）+ 看完电影喝东西，涵盖过去描述、邀约、商定时间全流程。

核心语法

1. 形容词谓语句 敬体过去时活用

在第9课非过去时基础上，新增过去时变化，分肯定/否定，为核心重点：

形容词分类 过去肯定式 过去否定式 例句（高い/静か）
Ⅰ类（い结尾） Aい→かった+です Aい→く+ありませんでした 高かったです / 高くありませんでした
Ⅱ类（形容动词） AⅡ+でした AⅡ+では(じゃ)ありませんでした 静かでした / 静かじゃありませんでした
  • 例句：昨日の映画はおもしろかったです。（昨天的电影很有趣）；試験は難しくありませんでした。（考试不难）

2. 否定形式替换：ありません = ないです

  • 核心规则：「ない」是Ⅰ类形容词，可替代「ありません」表否定，语气礼貌，过去时同步替换
    • 现在否定：Aくありません = Aくないです
    • 过去否定：Aくありませんでした = Aくなかったです
  • 拓展：名词谓语句也适用，如：学生ではありません = 学生ではないです

3. 形容词的中顿（连用形）

  • 用法：多个形容词描述同一事物，表并列顺接，用中顿形式连接，时态由句尾谓语决定
  • 变化规则：
    1. Ⅰ类形容词：Aい→く + て
    2. Ⅱ类形容词：AⅡ + で
  • 例句：映画の主人公は背が高くて、かっこいいです。（电影主人公又高又帅）；昨日の試験は複雑で、難しかったです。（昨天的考题又复杂又难）
  • 关键：中顿形式仅由前一个形容词词性决定，与后接词无关。

4. N まで

  • 用法：接名词表终点，可接时间/地点名词，表“到……为止”
  • 例句：昨日、遅くまで日本語の試験でした。（昨天日语考试考到很晚）；授業はここまでです。（今天的课就到这里）

5. 全然 + 形容词（否定）

  • 用法：程度副词，后接否定式，表完全不……，语气比「あまり」更强
  • 规则：可单独用「全然」表否定，口语常用
  • 例句：試験は全然難しくありませんでした。（考试一点都不难）；A:難しかったですか？B:ううん、全然。（难吗？/一点都不）

6. N とか

  • 用法：接名词表举例，口语专用，可单独用/多次连用
  • 规则：可省略主题/宾语助词「は/が/を」
  • 例句：ミルクティーとか、コーラとかが好きです。（喜欢奶茶、可乐之类的）；木曜日の午後以降とかはどうですか。（周四下午以后之类的时间怎么样？）

重点表达

1. 时刻读法

整点（～時）

  • 1時→いちじ、4時→よじ、7時→しちじ、0時→れいじ
  • 疑问词：何時(なんじ)

分钟（～分/～半）

  • 特殊读音：5分→ごふん、10分→じっぷん、30分→さんじっぷん
  • 半：～時半(はん)，如：4時半→よじはん
  • 疑问词：何分(なんぷん)

2. 邀约的2种核心方式

方式1：N は どうですか（提议）

  • 用法：无动词，直接用提议内容作主语，表“……怎么样？”，适用于定时间/定事项
  • 例句：来週はどうですか。（下周怎么样？）；6時はどうですか。（6点如何？）
  • 辨析：第9课中此句式表询问看法，本课时表提议，根据语境判断。

方式2：(一緒に) N を しましょうか

  • 用法：接具有动作意义的名词，表“一起做……吧？”，邀约专用
  • 例句：映画の後、一緒にお茶をしましょうか。（看完电影一起喝点东西吧？）
  • 注意：「お茶をします」表“喝点东西”，为日语固定表达。

3. 对邀约的回应：赞成/拒绝

赞成（礼貌+口语）

  • いいですね。（好啊）
  • はい、大丈夫です。（好的，没问题）
  • そうしましょう。（就这么做吧）
  • 拓展：可加「ありがとうございます」感谢对方邀约。

拒绝（日语委婉表达，核心：不直接说“不”）

  • 大丈夫です。ありがとうございます。（不用了，谢谢）
  • すみません。ちょっと……今度はぜひ一緒に。（不好意思，我不太方便，下次一定一起）
  • けっこうです。（不用了）
  • 关键：拒绝时可搭配歉意词/说明后续邀约，符合日语表达习惯。

4. 口语常用回应语

表达 释义 适用场景
よかったですね 太好了/真好呀 为对方的好事感到高兴，「よかった」是「いい」过去式
まあ、なんとか 还凑合/勉强可以 回应对方的询问（如考试/工作情况），表结果一般
うん/ううん 嗯/不不 「はい/いいえ」的口语形式，仅用于亲密关系，不用于上级
あら 哎呀 女性用感叹词，表惊讶，男性用「え？」替代
